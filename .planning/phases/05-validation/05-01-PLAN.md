  <name>Task 3: Performance Analysis and Roadmap Update</name>
  <files>.planning/ROADMAP.md</files>
  <action>
    Run the following SQL queries to verify VAL-01 success criteria and update the ROADMAP.md status.

    Query 1: Daily P&L over last 14 days
    ```sql
    -- Note: This assumes realized P&L is tracked by joining entry/exit in a simplified way 
    -- or by looking at exit records if they contain enough info.
    SELECT 
        decision_time::date as trade_date,
        SUM(CASE 
            WHEN direction = 'long' THEN (exit_price - entry_price) * quantity
            WHEN direction = 'short' THEN (entry_price - exit_price) * quantity
            ELSE 0 
        END) as daily_pnl
    FROM trade_log
    WHERE action IN ('HARD_STOP_LOSS', 'TRAILING_STOP', 'TAKE_PROFIT', 'SIGNAL')
    GROUP BY trade_date
    ORDER BY trade_date DESC;
    ```

    Query 2: Coverage of exit reasons
    ```sql
    SELECT exit_reason, count(*) 
    FROM trade_log 
    WHERE action IN ('HARD_STOP_LOSS', 'TRAILING_STOP', 'TAKE_PROFIT', 'SIGNAL')
    GROUP BY exit_reason;
    ```

    Query 3: Presence of risk/sentiment metadata in entries
    ```sql
    SELECT 
        COUNT(*) as total_entries,
        COUNT(sentiment_score) as with_sentiment,
        COUNT(regime_at_decision) as with_regime,
        COUNT(vix_at_decision) as with_vix
    FROM trade_log 
    WHERE action IN ('BUY', 'SHORT');
    ```

    Update ROADMAP.md:

    - Update VAL-01 in Requirements to 'Complete'.
